A custom WordPress Block that pull in latest five nursing jobs depdent on critera.

== Description ==

The plugin works but entering the API urls within the plugin settings.

Then add the block to a page and use the filters on the right sidebar.

The block will fire a request to the custom WP-API created in the plugin.

When this route is it, it proxies the external data via PHP request and returns the data to be filled into the custom rest route.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/ih-top-nurse-jobs` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
